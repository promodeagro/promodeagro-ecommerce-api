================================================================================
  COMMON TASKS
================================================================================

1. Check LocalStack Status
   $ curl http://localhost:4566/_localstack/health

2. List DynamoDB Tables
   $ aws dynamodb list-tables \
     --endpoint-url http://localhost:4566 \
     --region ap-south-1

3. Test API Endpoint
   $ curl http://localhost:4000/product

4. Create Product
   $ curl -X POST http://localhost:4000/product \
     -H "Content-Type: application/json" \
     -d '{"id":"1","name":"Test","price":100}'

5. Get All Products
   $ curl http://localhost:4000/product

6. Stop All Services
   $ bash scripts/stop-localstack.sh

7. Clean Up Everything
   $ docker-compose -f doc/localstack/docker-compose-localstack.yml down -v

================================================================================
  TROUBLESHOOTING
================================================================================

Problem: Port already in use
  $ lsof -i :4566
  $ kill -9 <PID>

Problem: LocalStack won't start
  $ docker ps
  $ docker-compose -f doc/localstack/docker-compose-localstack.yml logs localstack

Problem: Tables not created
  $ bash scripts/create-tables.sh

Problem: API connection issues
  $ curl http://localhost:4000/product

Problem: Everything broken
  $ docker-compose -f doc/localstack/docker-compose-localstack.yml down -v
  $ bash scripts/start-localstack.sh
